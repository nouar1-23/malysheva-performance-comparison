# Сравнение производительности: методы критерия дихотомии

Данный репозиторий содержит набор программ для сравнительного анализа времени выполнения различных условий в цикле построения спектральных портретов (метод Малышевой).

## Описание
В коде реализованы три различных варианта проверки условия в итерационном цикле. Цель данного исследования — сравнить вычислительную эффективность каждого подхода при различных размерах матриц ($n$) и диапазонах параметров ($r$).

## Как использовать
1. Откройте **MATLAB**.
2. Поместите все файлы проекта в рабочую директорию.
3. Для визуализации результатов и построения сравнительных графиков введите в командной строке имя функции, указав конечное значение размера матрицы $n$ (например, 200), для которого необходимо отобразить результаты:
   ```matlab
   tim(200)
4. Программа выполнит расчеты для каждого из трех вариантов.
5. В конце будет построен график, наглядно показывающий, какой метод работает быстрее.
